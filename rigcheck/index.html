RigCheck In development

Serial CAT communication diagnostics using Hamlib. Verify your radio connection before it matters.

Watch on GitHub

Status

RigCheck is currently in development. No release is available yet.

Watch the GitHub repository for announcements, or follow the project on the ShackDesk home page.

Planned Features
CAT Diagnostics

Test serial communication with your radio using Hamlib's rigctl. See exactly what's working and what isn't.

Connection Verification

Confirm your rig is connected and responding before an operating session or EMCOMM activation.

Hamlib Integration

Built on the same Hamlib library used by WSJT-X, JS8Call, and other digital mode software.

Offline-First

No internet connection required. Works anywhere PortPane does.

In the meantime

Looking for a tool that's available now? PortPane handles audio device and COM port management for digital mode operators and is available for download today.
