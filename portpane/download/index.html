How to Download PortPane

PortPane is in alpha testing and is not yet code-signed. Unsigned apps trigger warnings in browsers and Windows before they will run. This page walks you through each step.

Why does this happen? Windows SmartScreen and browser download protection flag any executable that does not have a recognized publisher certificate. PortPane's code signing certificate is pending. In the meantime, you can safely proceed — all official builds are produced by automated GitHub Actions workflows from the public repository and include a SHA-256 hash for verification. See the code signing policy for details.

Step 1 — Download the installer

Click the button below to download the latest alpha installer. The file is served directly from GitHub Releases.

Download PortPane Alpha Installer

File: PortPane-win-Setup.exe

Windows 10 or Windows 11  ·  64-bit  ·  No admin rights required  ·  Installs per-user

Your browser will likely show a warning. Continue to the section for your browser below.

Firefox and most other browsers show a simpler warning. Look for a prompt asking whether to save the file — choose Save or Save File. If a warning appears after saving, look for a Keep or Allow option.

Once the file is saved, open your Downloads folder and run it. Continue to Step 3 for the Windows SmartScreen prompt.

Step 3 — Windows SmartScreen prompt

When you run the installer, Windows Defender SmartScreen shows a blue dialog. The option to continue is hidden by default — you have to reveal it.

1
A blue dialog appears: "Windows protected your PC"

The dialog shows the app name and publisher (listed as Unknown). There is no Run button visible yet.

This is a small text link in the body of the dialog, not a button. It is easy to miss.

Click Run anyway

PortPane installs and launches.

Screenshot — Windows 11 SmartScreen after More infoShows "Run anyway" button.

No "Run anyway" option? Some clean installs of Windows 11 have Smart App Control enabled, which is more restrictive than SmartScreen and does not show a "Run anyway" option at all.

To work around this: right-click the downloaded PortPane-win-Setup.exe, choose Properties, tick the Unblock checkbox near the bottom of the General tab, click OK, then run the file again.

Screenshot — Windows file Properties dialogThe General tab shows a security message at the bottom with an "Unblock" checkbox. The checkbox is unchecked by default.
After installation

PortPane installs per-user to %LocalAppData%\PortPane\. No administrator rights are required.

A Start Menu shortcut is created automatically. Future updates are delivered in-app — you will see a notification in PortPane when a newer alpha build is available and can apply it with one click.

To uninstall: open Settings → Apps and find PortPane in the list, or run the uninstaller from %LocalAppData%\PortPane\.

Verifying the download

Every official PortPane release includes a SHA-256 hash file alongside the installer. You can verify the download using PowerShell:

Get-FileHash PortPane-win-Setup.exe -Algorithm SHA256 | Select-Object Hash

Compare the output against the .sha256 file published on the GitHub Releases page.

All official builds are produced by GitHub Actions from the public source repository. See the code signing policy for the full picture.
